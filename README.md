# 📚 Sagopa (교내 도서관 웹 사이트)

Next.js, NestJS, Python으로 구현한 도서 관리 및 중고 거래 통합 시스템

---

### ✨ 프로젝트 개요 및 목표
본 프로젝트는 도서관 관리 시스템과 학생 간의 중고 도서 거래 마켓을 통합한 종합 서비스입니다. Next.js로 고성능 $\text{UI}$를 구축하고, NestJS를 메인 $\text{API}$ 서버로, Python을 보조 백엔드로 활용하는 다중 언어 백엔드 아키텍처를 구현하여 기술적 숙련도를 극대화합니다.
| 항목 | 내용 |
| :--- | :--- |
| **프로젝트 목표** | 도서 대출/반납 시스템과 **학생 간 중고 도서 거래 플랫폼**을 통합한 종합 서비스 제공. |
| **핵심 아키텍처** | **Next.js**와 **NestJS/Python**을 분리한 **다중 언어 백엔드(Polyglot Backend)** 아키텍처 구축. |
| **기술적 도전** | **Next.js**를 활용한 $\text{SSR/SSG}$ 구현 및 **NestJS/Python** 간의 유기적인 $\text{API}$ 연동. |
| **로직 난이도** | **중고 거래 트랜잭션**과 도서 대출/반납 로직의 **데이터 무결성** 확보. |
| **강조 역량** | **NestJS/Python** 학습 및 **Docker**를 활용한 복잡한 환경 관리 능력 증명. |
---

### 💻 기술 스택 (Tech Stack)

| 분야 | 핵심 기술 | 사용 목적 |
| :--- | :--- | :--- |
| **프론트엔드** | **Next.js (TS) + React** | **SSR/SSG** 및 **TypeScript**를 통한 안정적인 UI 구현. |
| **백엔드 (주)** | **NestJS (TS)** | **주요 RESTful API 서버**. Spring과 유사한 구조로 설계된 Node.js 백엔드 학습. |
| **백엔드 (보조)** | **Python** | 데이터 분석 또는 특정 로직 처리 등 **보조적인 백엔드 역할** 통합 경험. |
| **데이터베이스** | **MariaDB** | 도서 정보, 대출 기록, 중고 거래 목록 관리. |
| **환경 관리** | **Docker** | 전체 개발 환경 컨테이너화 및 환경 일관성 유지. |

# ⚙️ 핵심 구현 기능

| 구분 | 상세 기능 |
| :--- | :--- |
| **중고 거래 트랜잭션** | 중고 도서의 상태 변경 등 **데이터 무결성**이 중요한 로직을 NestJS로 구현. |
| **다중 백엔드 연동** | Next.js가 NestJS와 Python 서버를 유기적으로 호출하는 **API 게이트웨이** 구조 설계. |
| **도서 관리** | 도서 대출/반납, 연체 관리 등 도서관 핵심 기능 구현. |
| **검색 최적화** | Next.js의 기능을 활용하여 도서 및 거래 게시물에 대한 **SEO** 최적화. |
