# 📚 Sagopa (교내 도서관 웹 사이트)

**Next.js와 Spring Boot로 구현한 고성능, 보안 강화 도서 및 중고 거래 통합 관리 시스템**

---

### ✨ 프로젝트 소개 및 고도화 목표

'Sagopa'는 대학 도서관의 **도서 관리 시스템**과 **교내 중고 도서 거래 플랫폼**이 통합된 웹 서비스입니다. **Next.js와 TypeScript**를 통해 현대적이고 효율적인 $\text{SPA}$ (Single Page Application)를 구축하고, **Spring Boot**의 강력한 $\text{REST API}$와 $\text{MariaDB}$를 활용하여 **복잡한 거래 로직과 견고한 인증 시스템**을 안정적으로 처리합니다.

**고도화 목표:**
* **서비스 통합:** 도서 대출 관리 시스템과 사용자 간의 중고 거래 플랫폼을 하나의 서비스로 통합.
* **거래 안정성:** $\text{Spring Boot}$의 트랜잭션 관리를 통해 중고 거래의 **등록, 구매, 상태 변경** 등 복잡한 과정을 안정적으로 처리.
* **보안 강화:** $\text{Spring Security}$를 활용하여 **학생, 관리자**의 권한은 물론 **중고 거래 판매자**와 **구매자**의 역할을 명확히 통제.

**주요 기능:**
* **도서 검색/대출 관리:** 도서관 소장 도서의 검색, 대출/반납 로직 처리.
* **🆕 중고 도서 거래 마켓:** 사용자가 직접 교재 및 도서를 등록하고, **실시간으로 조회 및 거래**할 수 있는 게시판 및 거래 시스템 구현.
* **관리자 시스템:** 도서 등록, 사용자 관리, **중고 거래 모니터링** 및 관리자 전용 기능.
* **인증/권한:** $\text{JWT}$ 기반 인증 및 $\text{Spring Security}$를 통한 역할 기반 $\text{Access Control}$ 구현.

---

### 💻 기술 스택 (Tech Stack)

이 프로젝트는 $\text{Next.js}$와 $\text{Spring Boot}$가 독립적으로 운영되는 **분리형 아키텍처**입니다. 

#### 📦 프론트엔드 (Frontend - Next.js & TypeScript)
| 기술 | 설명 |
| :--- | :--- |
| **Next.js** | 라우팅 및 초기 렌더링을 담당하는 React 프레임워크. |
| **TypeScript** | 관리 및 거래 시스템의 복잡한 데이터 구조의 안정성 확보. |
| **React** | 사용자 인터페이스 구축. |
| **State Management** | $\text{Redux Toolkit}$ 또는 $\text{Recoil}$ 등을 활용한 전역 상태 관리. |

#### ⚙️ 백엔드 (Backend - Spring Boot & Java)
| 기술 | 설명 |
| :--- | :--- |
| **Spring Boot** | **안정적인 RESTful API 서버** 및 핵심 비즈니스/거래 로직 구현. |
| **Java** | 백엔드 로직 구현. |
| **Spring Security & JWT** | 사용자 및 관리자의 **권한 기반 인증/인가** 처리. |
| **Spring Data JPA** | 도서, 대출 기록, **중고 거래 테이블** 등 데이터베이스 접근 및 $\text{ORM}$ 구현. |

#### 💾 데이터베이스 및 인프라
| 기술 | 설명 |
| :--- | :--- |
| **MariaDB** | 도서 정보, 대출 기록, **중고 거래 목록 및 거래 상태** 저장. |
| **Docker** | $\text{Next.js, Spring Boot, MariaDB}$ 환경을 독립적인 컨테이너로 통합 관리. |
